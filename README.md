-- Program za random izvlačenje pitanja --

Prilikom dodavanja novih pitanja naophodno je da svako novo pitanje počinje u novom redu, i završava u istom tom redu.
Moguće je ubacivati pitanja za bilo koji predmet, kao i neograničen broj pitanja.
Pitanja se dodaju u fajl pitanja.txt.
